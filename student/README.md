# 🧑‍🎓 เริ่มตรงนี้ — สำหรับนักเรียน

ยินดีต้อนรับสู่ Edge AI Workshop! วันนี้ทีมของน้องจะสอน AI ให้ "มอง / ฟัง / รู้สึก" สิ่งรอบตัว แล้วเอาไปรันบนบอร์ดจริง

## วันนี้ทำอะไร

- **เช้า:** ทำให้ UNO Q เป็นของทีม + ต่อ input ครบ 3 แบบ (sensor / กล้อง / ไมค์)
- **บ่าย:** เก็บข้อมูล → เทรน model → ลงบอร์ด → ทดสอบ → ส่งงาน
